Marseille Public Wifi Map

Home / portfolio / Marseille Public Wifi Map

Carte des points Wi-fi de Marseille

Traiter des données publiques & les afficher sur une carte.

But :

Mobiliser des données publiques récupérées en ligne, et les afficher sur une carte interactive.

Données récupérées sur opendata.regionpaca.fr

Technologies :

Site réalisé en Javascript, HTML, CSS, Bootstrap et git. La librarie JavaScript Leaflet et l'API mapbox ont été utilisées pour la carte. Les données publiques étant souvent au format CSV, XLS ou encore JSON, j'ai utilisé D3.js (https://github.com/d3/d3-request) pour les manipuler.

Interactivité :

La carte affiche des marqueurs cliquables. Lorsque l'utilisateur clique sur un marqueur, il obtient des informations sur le lieu lié au marqueur (Points wifi publics de Marseille).

  • Mapbox
  • Leaflet
  • d3.js
  • Javascript
